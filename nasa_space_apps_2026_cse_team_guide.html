১. Demo Video-ই আসল weapon
Judge-রা হাজার project দেখেন। ২–৩ মিনিটের compelling video-ই তোমাকে আলাদা করবে। Screen recording নয় — live demo + narration + impact statement থাকতে হবে। Video-তে "বাংলাদেশের X জন মানুষ এই সমস্যায় ভোগেন" এই ধরনের real impact বলো।
২. "Made with NASA data" শুধু বললে হবে না
NASA data কীভাবে specifically ব্যবহার হয়েছে সেটা submission-এ বিস্তারিত লিখতে হবে। "আমরা GPM IMERG V06 data ব্যবহার করে ০.১° resolution-এ Bangladesh-এর precipitation pattern analyze করেছি" — এই ধরনের technical specificity judge-রা খোঁজেন।
৩. Working prototype মাস্ট — "concept only" জেতে না
২০২৫ সালের সব বিজয়ী কাজ করা prototype জমা দিয়েছে। শুধু idea pitch কখনো গ্লোবাল জেতেনি। Minimum viable product: data ফেচ করছে, কিছু দেখাচ্ছে, live URL আছে। Perfect না হোক, কিন্তু চালু থাকতে হবে।
৪. Local judge → Global judge connection তৈরি করো
Local event-এ ভালো করলে Global Nominee হওয়ার চান্স বাড়ে। Local judge-দের সামনে presentation-এ energy ও passion দেখাও। Bangladesh context টা strong ভাবে তুলে ধরো — judge-রা local relevance পছন্দ করেন।
৫. Open Source + Good README = Extra points
NASA open source culture পছন্দ করে। MIT license দাও। README-তে: Problem, Solution, NASA data used, How to run, Screenshots, Team members। GitHub এ stars থাকলে আরো ভালো। Submission page-এ Project Description ১০০০+ শব্দ লেখো।
